# MultiClient-Chat-Application
I built a Java-based multi-client chat system using socket programming where a server handles multiple clients using multithreading. Each client runs independently, and messages are broadcasted in real-time using a thread-safe shared client list. I also built a Swing GUI for the client-side interface.

Skills:
✔ Java
✔ Socket Programming
✔ TCP/IP
✔ Multithreading
✔ Concurrency
✔ Client-Server Architecture
✔ Real-time communication
✔ Swing GUI
✔ Message broadcasting

Compilation:
-> open project
-> cd src
-> javac server\*.java client\*.java
-> java server.ChatServer
-> open new terminals
-> again cd src
-> java client.ChatClient
-> java client.ChatClientGUI

Will show entire UI with chatting between two or more clients.
